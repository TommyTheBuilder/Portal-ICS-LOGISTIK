Template-Designer einrichten

Der integrierte Build unter apps/web/dist ist auf diesem System aktuell nicht vorhanden.

Du kannst hier eine erreichbare Designer-URL hinterlegen (z. B. interner Host, Dev-Server, eigener Deploy). Die URL wird im Browser gespeichert.

Zurück zur Admin-Seite
